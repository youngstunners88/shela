# StoryChain Workspace - AGENTS.md

> Load context from current directory's AGENTS.md

## Quick Access
- Skills: See `Skills/*/SKILL.md` for specific tools
- Projects: Each project has its own AGENTS.md
- Memory: Use `zo-memory` skill for persistent context

## Default Behavior
1. Check current directory for AGENTS.md or SOUL.md
2. Load most specific context available
3. Fall back to directory parent if needed
