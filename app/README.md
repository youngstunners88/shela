
```bash
npm run build
```

## About Bhubezi

Bhubezi (meaning "lion" in isiZulu) is a community-powered app for Johannesburg's minibus taxi industry.

### Features
- 📍 Real-time taxi tracking
- 💰 Community-verified prices
- 💬 Route chat
- 🏆 Rewards & gamification
- 📱 PWA support
